# C# Development Portfolio & Coursework

This repository contains the core projects and technical concepts I developed during my C# development course. 

Key Projects Completed
1. Web Application
Description: A dynamic and responsive web application designed to handle secure user interactions and data rendering.
Key Tech Used: C#, ASP.NET Core, MVC Architecture, Razor Pages.
What I Learnt:** State management, routing, and building scalable server-side web logic.

2. Car Insurance Database
Description: A relational data management system built to process customer profiles, vehicle details, and policy quotes.
Key Tech Used: C#, ADO.NET, SQL Server, Stored Procedures.
What I Learnt: Database normalization, executing secure CRUD operations, and managing relational data connections.

3. 21 Card Game
Description: A digital implementation of the classic Blackjack card game featuring automated dealer logic and score tracking.
Key Tech Used: C# Console / Desktop application, Collections (Lists/Arrays), Randomization algorithms.
What I Learnt: Core Object-Oriented Programming (OOP) principles like encapsulation, inheritance, and handling game-state logic.

4. Code First Database
Description: A database deployment approach where the relational schema is generated dynamically from C# model classes.
Key Tech Used: C#, Entity Framework Core (EF Core), LINQ, Data Migrations.
What I Learnt: Automating database updates through C# models, handling object-relational mapping (ORM), and running database migrations.

Technical Skills Gained

Language Proficiency: C# (Object-Oriented Programming, Collections, Control Flow)
Frameworks & Architecture: .NET Core, ASP.NET Core, MVC
Data & Databases: Entity Framework Core (Code-First), LINQ, SQL Server, Relational Databases
Tools & Workflows: Git/GitHub, Visual Studio / VS Code, Migrations and Schema Tracking

Course Highlights
Throughout this course, I focused on writing clean, type-safe, and maintainable C# code while adhering to modern development practices and structural design patterns.
